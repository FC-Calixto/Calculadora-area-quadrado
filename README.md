# Calcula Área do Quadrado

Projeto simples em Java que solicita ao usuário a medida do lado de um quadrado, valida a entrada garantindo até duas casas decimais, e calcula a área exibindo o resultado formatado.

## Funcionalidades

- Solicita a medida do lado do quadrado em metros, aceitando números decimais com até duas casas decimais.
- Normaliza entrada substituindo vírgula por ponto para facilitar a digitação.
- Valida entrada para garantir que seja número positivo e formato correto.
- Exibe a área do quadrado com duas casas decimais, seguindo o padrão métrico (m²).
- Trata entradas inválidas com mensagens de erro amigáveis.

## Como usar

1. Clone ou baixe este repositório.
2. Compile o programa: javac CalculaAreaQuadrado.java
3. Execute o programa: java CalculaAreaQuadrado
4. Informe a medida do lado quando solicitado.

## Tecnologias

- Java 8 ou superior
- Biblioteca padrão Java (java.util.Scanner)
